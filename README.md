# Smart_Cage_Android
(2022 univ lecture)

- 로그인
- 회원가입
- 유저의 파충류 종 list 표시 (data from server)
- 유저의 특정 파충류에 해당하는 cage list 표시 (data from server)
- 특정 cage 정보 확인
- 특정 cage의 센서 정보 상세 확인
- 사육 파충류 그룹 생성
- 케이지 생성
- 유저 맞춤 센서 범위 설정
- 센서 범위 설정 Default 값 불러오기
- 케이지 ae의 센서값 설정을 위한 Control Cnt 생성
- 사용자가 설정한 센서 값 전송
- 알림 (범위 내 센서 값 확인)
- 개인 정보 관리
- 로그아웃

+ 추가적인 편의 기능
